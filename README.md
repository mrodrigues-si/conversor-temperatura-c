# Conversor de Temperatura em C 🌡️

Conversor simples de Celsius para Fahrenheit feito em linguagem C durante o 1º semestre de Sistemas de Informação na UFU.

## 📝 Funcionalidades
- Recebe temperatura em Celsius via teclado
- Converte automaticamente para Fahrenheit usando a fórmula: `(C × 9/5) + 32`
- Exibe o resultado formatado com 1 casa decimal

## 🛠️ Tecnologias
- Linguagem C
- Code::Blocks

## 🎯 O que aprendi
- Uso de variáveis do tipo `float` para números decimais
- Leitura de dados com `scanf` e exibição com `printf`
- Operações matemáticas com prioridade de parênteses
- Formatação de saída (`%.1f`)
